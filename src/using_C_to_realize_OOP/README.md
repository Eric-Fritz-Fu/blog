# 用 C 语言实现面向对象

## 引言

尽管我们通常会说 C 语言是一门面型过程的语言，但这并不意味着没法使用面向对象的思想。本质上，语言只是一种表达方式。而支持面向对象，只是是在设计编程语言时一个特定的功能，使其可以更好地表达面向对象的的程序。通过使用结构体、函数指针，C 语言也可以实现面向对象思想。

## 实现

面向对象编程的三大特点是封装、继承、多态，三者缺一不可，我们将会分别实现它们。
